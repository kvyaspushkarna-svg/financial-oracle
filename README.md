# Ramen to Riches (Financial Oracle)

A four-part interactive web app that teaches **first-time investors** the essentials of saving and wealth-building — using behavioural-finance ideas to ease the fears that stop people from starting.

## What it does
- Breaks investing basics into four simple, guided parts.
- Uses behavioural-finance concepts to address the mental barriers new investors face.
- Keeps the language beginner-friendly, practical, and jargon-free.

## Built with
Interactive web app · Behavioural finance concepts · Financial-literacy content

## Run it locally
```bash
# open the app in your browser
# (if static) simply open index.html
# (if a framework) install and start:
npm install
npm start
```

## Why I built it
I care about making complex things simple and useful. This started as a way to help beginners get past the fear of starting — because most people don't lose money investing, they lose it by never beginning.
